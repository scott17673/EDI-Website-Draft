# E D Industrial Website

Static brochure-style site for E D Industrial with three pages:

- `/`
- `/services/`
- `/contact/`

## Run locally

```bash
node server.mjs
```

Then open `http://localhost:3000`.

## Content updates

Most editable company content lives in:

- `assets/content.js`

## Design system

Global styling lives in:

- `assets/styles.css`

## Notes

- The logo file is currently a replaceable placeholder at `assets/logo-ed-industrial.svg`.
- Jeff Bullett and Scott Dumont currently route through the main sales inbox and phone number until direct public details are supplied.
- The contact form is static and opens the visitor's email app with a pre-filled inquiry.
